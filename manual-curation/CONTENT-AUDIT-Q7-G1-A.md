## 1) worlds/absolute-duo

### 现状判断
- `curated/README.md` 仍是**摘要级**，只有一句总述 + 源头说明，没有展开世界结构。
- `curated/curation-notes.md` 也偏**流程记录**，只列核心概念与低置信项，未补世界百科。
- `characters-index.json` 已有主要角色，但条目信息偏轻，仍明显缺“可 RP 化”的细颗粒描述。

### 角色缺口
重点人物目前普遍缺：
- **外表**：发色、身高、体型、标志性特征、第一眼辨识点。
- **服装**：校服/战斗服/私服差异，配色与道具细节。
- **气质/说话风格**：语气、情绪表达、压迫感/亲和力。
- **年龄与身份细化**：年级、教师/学生/管理者层级、公众身份。
- **能力表现**：焰牙战斗风格的具体动作、限制、战绩、代价。
- **关系动态**：三角关系、竞争、误会、师生张力、同伴磨合。
- **RP 钩子**：冲突开场、训练课题、搭档考验、复仇推进、校园日常事件。

### 世界设定缺口
- **地点**：昊陵学园内部区域、宿舍、训练场、研究设施、周边城市与出行节点。
- **组织**：学园管理层、研究/实验线、敌对组织、后期反派阵营与其上下级。
- **事件**：主线战役、训练考核、搭档测试、人工强化相关事件、重大转折点。
- **规则**：黎明星纹等级、焰牙判定逻辑、搭档制度、强化风险与制度约束。

### 自适应补全清单
1. 为主角群补全“外观 + 服装 + 气质 + 标志动作”。
2. 为焰牙/战斗条目补全“实战表现 + 代价 + 限制 + 招式场景”。
3. 为昊陵学园补全“地点层级图”。
4. 为学园相关组织补全“职能、关系、敌对/合作边界”。
5. 为后期敌对势力补全“阵营名、首脑、战术风格、出现事件”。
6. 将“复仇线 / 强化线”拆成可回放 RP 钩子，而不是只保留结论性摘要。

## 2) worlds/acg-character-database

### 现状判断
- 该目录**不是单一世界**，而是跨作品角色心理/类脑参考库，这一定位是正确的。
- `README.md` 已明确“非单一作品世界”，属于**库说明**，不是剧情世界百科。
- `curation-notes.md` 主要是门禁、决策、风险说明，覆盖“如何使用”，但不是内容扩展。
- `characters-index.json` 里 `charactersObserved` 很长，但本质是**观察名单**，不是可直接 RP 的角色卡集合；`characters` 为空是设计选择。

### 主要缺口
- **不是缺地点/组织/世界规则，而是缺每条观察项的结构化展开**。
- 当前只保留“角色名 + 来源作品”，对角色分析极薄。
- 若用于角色模型检索，仍缺：
  - 外表/服装模板
  - 气质/说话习惯
  - 年龄段/身份标签
  - 典型能力表现
  - 常见关系动态
  - 适用 RP 钩子
- 人格模板仅列名称，未见每个模板的定义边界、关键词、反例与适配条件。

### 适配性判断
- 因为它是数据库而非单一世界，**不应强行补地理、国家、时间线、统一组织**。
- 真正需要补的是“索引元数据”和“使用说明”，而不是世界观剧情。

### 自适应补全清单
1. 为 `charactersObserved` 中高频条目补“来源作品内角色卡摘要”。
2. 为人格模板补“定义、典型外显、常见误判、适配场景”。
3. 增加“来源作品 / 连续性 / 版本差异”字段，避免混用。
4. 若后续要支持 RP，增加“仅作角色原型参考”的使用示例。
5. 对重复或多版本条目补去重说明与版本差异说明。

## 3) worlds/akame-ga-kill

### 现状判断
- `curated/README.md` 已有世界核心设定：腐败帝国、革命军、帝具、夜袭、狩人。
- 但整体仍偏**总纲摘要**，更像“入门说明”，不是完整世界百科。
- `curation-notes.md` 主要记录阅读来源和风险，未补足设定细节。
- `characters-index.json` 已列主要人物与帝具，但角色信息依旧偏短。

### 角色缺口
当前人物条目普遍缺：
- **外表**：发色、体型、装束、武器外观、识别特征。
- **服装**：夜袭制服、帝国军装、狩人制服、帝具装备细节。
- **气质**：冷酷/热血/优雅/疯批/正义执念等可直接用于 RP 的描述。
- **年龄身份**：军阶、出身、在夜袭/狩人中的定位层级。
- **能力表现**：帝具实战效果、限制、副作用、对战例证。
- **关系动态**：塔兹米与同伴、赤瞳与黑瞳、艾斯德斯与狩人/帝国的张力。
- **RP 钩子**：潜入、处刑、追捕、训练、牺牲抉择、价值冲突。

### 世界设定缺口
- **地点**：帝都各区、夜袭据点、革命军据点、边境/战场、实验与处刑场所。
- **组织**：革命军内部结构、帝国中枢、狩人编制、贵族/警备系统、艾斯德斯麾下关系网。
- **事件**：帝具争夺、夜袭任务链、狩人围剿、帝都政变、关键死亡节点。
- **规则**：帝具适格/代价、危险种来源、帝具数量与稀有性、战斗晋级规则、帝国统治逻辑。

### 自适应补全清单
1. 为夜袭与狩人主力人物补“外表/服装/气质”三件套。
2. 为每件帝具补“形态、限制、代价、标志性战斗表现”。
3. 为帝都与革命军补“地点分层 + 行动场景”。
4. 为帝国、革命军、夜袭、狩人补“组织结构与相互关系”。
5. 为主线事件补“时间顺序 + 可复用冲突点”。
6. 为塔兹米 / 赤瞳 / 艾斯德斯 / 黑瞳等核心人物补高密度 RP 钩子。

## 总体结论

### 完整度分级
- `absolute-duo`：**中低完整度**，人物和设定均需扩写，尤其缺外表/服装/气质/组织与规则。
- `acg-character-database`：**结构正确但内容不应按单世界补全**，应补索引元数据与原型定义，而非世界观。
- `akame-ga-kill`：**中等偏低完整度**，世界总纲已成型，但人物与设定仍需要可 RP 化扩展。

### 优先补全顺序
1. 角色外表 / 服装 / 气质。
2. 角色关系动态与 RP 钩子。
3. 世界地点与组织结构。
4. 能力表现、限制与代价。
5. 关键事件与规则说明。
