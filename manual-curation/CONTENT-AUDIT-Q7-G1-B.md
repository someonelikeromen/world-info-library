## 审计范围与方法

已逐个读取指定世界的：

- `curated/README.md`

---

## 总体结论

三组世界的 curated 层都具备基础设定骨架，但人物外观字段普遍缺失：

| 世界 | 人物索引状态 | 外观/服装字段完整度 | 扩充优先级 |
|---|---|---:|---|
| `black-bullet` | 仅 5 名核心人物，战斗/阵营信息有，但人物数与细节偏少 | 极低：无外貌、服装、年龄段、气质视觉描述字段 | 高 |
| `blue-archive` | 20 个条目，但混合固定姓名、组织岗位、RP 槽位；非完整角色库 | 极低：无学生光环、制服、武器外观、学院视觉规范 | 最高 |
| `bocchi-the-rock` | 5 名核心人物，乐队关系清晰 | 低：性格较清楚，但无发型/服装/舞台形象/说话方式字段 | 中高 |

共性建议：后续扩充时为每个可扮演角色增加统一字段：

- `appearance`：身高体态、发型发色、眼睛、显著标识。
- `outfit`：日常服、制服、战斗服/舞台服、常见配饰。
- `demeanor`：第一眼气质、肢体语言、社交距离。
- `voiceStyle`：语气、口癖、称呼习惯、情绪波动表现。
- `rpHooks`：适合开场场景、冲突触发点、关系推进方式。
- `safetyNotes`：涉及未成年人、歧视、暴力、心理健康时的运行门禁。

- `worlds/black-bullet/curated/characters-index.json`
- 辅助：`world.json`、`knowledge-graph.json`、`relationship-graph.json`

### 当前内容概况

README 标明该世界为 Batch B 手工初版，主来源 `src-black-bullet-001`，原报告概况有人物 82、地点 87、道具 73。curated 重点为原肠动物、受诅之子、民警搭档、巨石碑与东京区域。

`characters-index.json` 当前仅收录 5 名人物：

1. `rentaro-satomi` / 里见莲太郎
2. `enju-aihara` / 蓝原延珠
3. `kisara-tendo` / 天童木更
4. `seitenshi` / 圣天子
5. `kagetane-hiruko` / 蛭子影胤

关系图覆盖莲太郎—延珠、木更、圣天子、蛭子影胤等主轴。世界规则已覆盖原肠动物病毒、受诅之子、錵/瓦拉尼姆武装、巨石碑防线、侵蚀率资源。

### 主要完整度缺口

#### A. 人物外观/服装/气质：严重缺失

当前人物条目有 `summary`、`roles`、`factions`、`abilities`、`items`、`relationships` 等字段，但没有专门外观字段。以下字段均缺：

- 发色、瞳色、发型、体型、年龄段/外观年龄。
- 日常服、战斗服、制服、政治礼服等服装描述。
- 武器/义肢/面具等视觉锚点的具体呈现。
- 第一眼气质：疲惫、危险、稚气、冷峻、圣洁、复仇感等。
- 说话风格、称呼方式、典型行为姿态。

人物逐项缺口：

| 人物 | 当前有内容 | 关键缺口 | 优先级 |
|---|---|---|---|
| 里见莲太郎 | 促进者、机械义肢、錵武装、与延珠搭档 | 义眼/义肢外观规格、校服/战斗装、少年气质与战斗姿态、枪械/刀具外观 | 高 |
| 蓝原延珠 | 兔型因子、受诅之子、超高速踢击、侵蚀率风险 | 外观年龄、兔型因子的视觉表现、服装、天真与战斗时反差、未成年人安全门禁 | 高 |
| 天童木更 | 社长、剑士、复仇线、天童式拔刀术 | 发型服饰、刀的样式、病弱/优雅/复仇气质、公司社长日常形象 | 高 |
| 圣天子 | 东京区域统治者、政治保护伞 | 正装/礼服、统治者视觉符号、公开演说姿态、温和与政治压力的表现 | 中高 |
| 蛭子影胤 | 面具男、改造战士、斥力场/枪械 | 面具细节、服装轮廓、危险气场、改造能力视觉效果、战斗台词风格 | 高 |

#### B. 人物覆盖量不足

README 指出原资料人物 82，但 curated 仅 5 名。作为 RP 世界，当前只能支持主轴危机，缺少支线与群像：

- 未收录其他民警组合/起始者与促进者。
- 未收录受诅之子社群代表、外围区居民、歧视者/保护者等社会冲突角色。
- 未收录政府、IISO、军警、反派势力的中层 NPC。
- 未收录原肠动物/黄道带级威胁的可运行条目细化。

#### C. 系统规则需要人物化落点

`world.json` 对侵蚀率、錵、巨石碑有规则，但人物索引中未把这些转化成运行字段：

- 起始者的动物因子、侵蚀率状态、情绪压力触发点。
- 促进者的武装配置、战术职责、与起始者的保护/依赖关系。
- 錵弹药和装备的补给限制。
- 社会歧视在具体场景中的表现边界。

### 自适应扩充清单

高优先级：

1. 为 5 名已有人物补齐 `appearance`、`outfit`、`demeanor`、`voiceStyle`、`rpHooks`。
2. 重点补莲太郎的义肢/义眼视觉规格、延珠的兔型因子与侵蚀率表现、木更的剑士/复仇气质、蛭子影胤的面具与改造能力视觉。
3. 增加未成年人/受诅之子 RP 安全门禁：禁止性化；歧视议题以保护、创伤、制度压迫为核心处理。
4. 将起始者-促进者搭档机制字段化：战斗职责、心理支撑、风险资源。

中优先级：

1. 扩充至少 10-15 个关键人物/角色槽位：其他民警、受诅之子代表、政府官员、IISO、外围区居民、反派协力者。
2. 为黄道带/阶段分类建立威胁条目，区分是否作为“角色”或“怪物/事件”。
3. 增加东京区域社会日常场景：学校、民警公司、外围区、巨石碑周边、政府会议。

低优先级：

1. 细化道具/武器视觉表：錵枪弹、刀具、义肢、面具。
2. 增加常用开局模板：委托任务、防线巡逻、受诅之子保护、政治护卫。

- 社团服装/岗位服装缺失：风纪、修女会、C&C、救援骑士团等。

### 当前内容概况

curated 层聚焦纽带乐队/结束乐队四名高中生在下北泽音乐场景中组队、练习、打工、登台并克服社交障碍与创作焦虑。

当前人物 5 名：

2. `ijichi-nijika` / 伊地知虹夏
3. `yamada-ryo` / 山田凉
4. `kita-ikuyo` / 喜多郁代

世界与关系层面较清晰：下北泽、STARRY、秀华高校、下北泽高中、乐队练习、展演厅文化、社交恐惧、音乐个性、社交平台分享。

#### A. 外观/服装/舞台形象：缺失

当前 `traits` 已记录主要性格，但没有外观字段。对现代乐队日常 RP 而言，外观、舞台站位、乐器与服装很重要。

逐项缺口：

| 人物 | 当前有内容 | 关键缺口 | 优先级 |
|---|---|---|---|

#### B. 口吻与演出运行信息不足

孤独摇滚的 RP 依赖日常对话和演出压力。目前缺少：

- 角色说话风格、吐槽节奏、沉默/退缩表现。
- 乐队练习流程：排练、写曲、售票、打工、Live 前后互动。
- 舞台表现：站位、紧张程度、演奏强项/短板。
- 喜剧化心理夸张的运行边界：notes 已说明不要当超自然能力，但还需具体示例。

#### C. 人物覆盖可适度扩充

当前 5 名已覆盖主轴，但下北泽音乐生态还可补：

- 其他乐队/前辈音乐人/观众/常客。
- STARRY 工作人员或演出相关 NPC。
- 后藤家庭成员、学校同学、粉丝/网络观众等日常支撑角色。

高优先级：

1. 为 5 名人物补 `appearance`、`outfit`、`instrumentVisual`、`stagePresence`、`voiceStyle`。
2. 为后藤独补“社恐表现边界”：喜剧夸张可用于心理描写，不当作超自然变形或能力。
3. 为四名乐队成员补排练/演出时的互补关系：谁推动、谁吐槽、谁救场、谁失误。
4. 补乐器与音乐职能：主音/节奏吉他区分、贝斯作曲、鼓手队长、主唱带动观众。
