# Q7A 全量内容完整度审计｜第1组（campione / cheng-long-adventures / chunibyo）

**审计目标**：通读指定世界的 `curated/README.md`、`curation-notes.md`、`characters-index.json`，并在必要时参考 `world.json`、`knowledge-graph.json`、`relationship-graph.json` 判断缺口。重点检查**人物外表 / 服装 / 气质**等字段完整度，输出明确补全清单。

**写入范围**：仅此审计报告，不修正文档。

---

## 总体结论

三套世界的 curated 层都已具备“世界主题 + 角色基础索引 + 关系/知识图谱”的骨架，但**人物层面的外观与穿着信息普遍缺失**，而且“气质/氛围/仪态”大多被 `traits`、`summary` 或 `notes` 近似替代，尚未形成可稳定检索和后续补写的结构化字段。

### 共同缺口

1. **缺少结构化外观字段**
   - 发色、瞳色、身高、体型、年龄感、面部特征、标志物等通常未单列。
2. **缺少结构化服装字段**
   - 校服、战斗服、便装、常驻配饰、标志性道具式穿搭未系统记录。
3. **缺少结构化气质字段**
   - 角色气场、说话风格、站姿/动作习惯、第一印象式的氛围描述未独立成项。
4. **现有字段偏重功能性**
   - `role`、`traits`、`abilities`、`rpUse` 能支撑剧情使用，但不足以支持外观检索、插画生成、角色对照或细颗粒人物复原。

## 世界一：`worlds/campione`

### 现状判断

- `README.md` 只说明本世界为手工索引初版完成，聚焦不从之神、弑神者、权能、魔术结社、神话相性。
- `curation-notes.md` 只说明权能的处理方式与 RP 风险。
- `characters-index.json` 目前只包含：`summary`、`roles`、`factions`、`locations`、`abilities`、`items`、`relationships`、`notes` 等。
- **完全缺少人物外观/服装/气质字段。**

### 重点缺口

#### 1) 草薙护堂 `godou-kusanagi`
- 已有：主角、弑神者、权能与关系。
- 缺少：外表、校服/常服、战斗时穿着、气质、少年感/压迫感等。
- 补全优先级：**高**。

#### 2) 艾丽卡·布朗特里 `erica-blandelli`
- 已有：骑士、魔术支援、魔剑/装备、伙伴关系。
- 缺少：标志性服装、贵族感、武装风格、外表识别点、气质锋利度。
- 补全优先级：**高**。

#### 3) 万里谷祐理 `yuri-mariya`
- 已有：媛巫女、灵视、净化。
- 缺少：巫女装束、温和/怯弱/端庄等气质、视觉识别点。
- 补全优先级：**高**。

#### 4) 雅典娜 `athena`
- 已有：不从之神、敌人与盟友的双重定位、神威。
- 缺少：神降临形态外观、服饰/神装、神性气场、形态变化差异。
- 额外风险：`notes` 只写“具体形态随剧情阶段变化”，但未拆分形态卡。
- 补全优先级：**高**。

#### 5) 罗濠 `luo-hao`
- 已有：武侠王、弑神者、威压。
- 缺少：武术宗师气场、服饰审美、体态、年龄感、宗师风范。
- 补全优先级：**中高**。

### Campione 明确补全清单

- 为每位核心角色新增结构化字段：
  - `appearance`
  - `outfit`
  - `aura` / `temperament` / `presence`
- 至少补足：
  - 发色、瞳色、体型、年龄感、典型表情
  - 常服 / 战斗服 / 仪式服 / 神显现形态服装
  - 角色气质关键词（例如沉稳、锋利、神秘、优雅、少年感）
- 对雅典娜建议拆成**不同显现形态**，避免把多阶段形象压成一个模糊条目。

- `appearance`
