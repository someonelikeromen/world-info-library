# ACG character database curation notes

## 手工阅读记录

- 阅读 `source-registry.json`，确认三个来源：2025年2月号、2025年3月号、咩咩 ACG 角色心理模型。
- 阅读 `characters/characters.json` 开头条目，确认其格式为跨作品 YAML/角色卡式心理资料，且含成人/NSFW 字段。
- 使用 `grep` 定位并人工复核 `comment` 列表，整理人物与人格模板清单。
- 阅读 `world/world.json` 开头，确认自动分类将角色条目重复归入 world，不能据此视为单一世界。

## 关键决策

- 明确本 slug 为“跨作品角色心理/数据库来源”，不是单一世界。
- 人物条目保留名称和来源作品；不保留露骨身体、性暴力、成人场景文本。
- 对重复出现的 2月/3月条目保留为“重复/多版本 observed”，不强行合并细节。
- 人格原型（傲娇、御姐、三无等）不列为人物，而列为 archetypes。

## 风险

- 原始库巨大且含重复；本次 characters-index 以手工复核 comment 清单为主，可能仍需后续去重、版本比对与来源规范化。
- 成人作品来源名称保留为索引事实，不代表纳入露骨内容。

## 验证状态

未运行机器 JSON 校验；JSON 为人工编写。

## 运行时门禁补充

- 本 slug 仅作为跨作品角色心理/类脑参考库运行，不应被当作单一世界或统一连续性。
- 调用时必须提供外部作品/continuity；否则只允许使用非露骨人格、语气、原型与来源标签。
- 不要自动合并不同作品条目的地理、阵营、时间线或关系。
