# 斩！赤红之瞳 curated 归档

人工整理入口。已阅读 `source-registry.json`、`world/world.json`、`characters/characters.json` 中的世界观、帝具、帝国、夜袭、狩人和主要人物条目。

## 核心设定
- 腐败帝国、革命军与帝都暗杀战构成主线。
- 帝具是初代皇帝以危险种素材和失传秘术打造的 48 件超凡武器，使用者需适格并承受代价。
- Night Raid（夜袭）是革命军部署在帝都的暗杀部队；Jaegers（狩人）是艾斯德斯率领、专门猎杀夜袭的帝国特殊警察部队。

## 人物重点
塔兹米、赤瞳、玛茵、希尔、布兰德、雷欧奈、娜洁希坦、拉伯克、须佐之男、切尔茜、艾斯德斯、黑瞳、威尔、赛琉、Dr.时尚、波鲁斯、兰、奥内斯特、小皇帝等。详见 `characters-index.json`。

## RP 使用
适合暗杀任务、帝具适格、帝都潜入、革命军据点、狩人与夜袭追捕、道德代价和角色牺牲。
