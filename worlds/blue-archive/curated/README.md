# 蔚蓝档案 / 基沃托斯 curated 归档

本目录整理 `blue-archive` 既有归档。原归档主要记录基沃托斯与学院自治设定，人物分类为空；因此人物清单以归档中明确提及的组织岗位、关键姓名与 Blue Archive RP 常用角色类型人工补全标注。

## 文件

- `world.json`：基沃托斯、学园自治、武装日常与主要学园。
- `source-registry.json`：来源登记。
- `characters-index.json`：学生、老师、组织岗位与关键人物清单。
- `knowledge-graph.json`：学园、联邦学生会、神秘、武装生态关系。
- `relationship-graph.json`：学院与组织关系。
- `curation-notes.md`：整理说明与风险。
