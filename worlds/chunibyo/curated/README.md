# 中二病也想谈恋爱 curated archive

手工整理对象：`chunibyo`。

本目录为 curated 层，只归纳既有归档中的原作向、SFW 世界观与 RP 可用结构；不修改 `raw/`、`imports/`、自动分类文件。

## 标准产物

- `world.json`：世界概览、主题、地点、组织、规则。
- `source-registry.json`：curated 使用的来源登记。
- `characters-index.json`：详细人物清单。
- `knowledge-graph.json`：世界知识节点与关系。
- `relationship-graph.json`：人物关系图。
- `curation-notes.md`：人工整理说明、取舍与风险。

## 整理重点

本世界以现代日本高中日常为底层现实，以“中二病”幻想语言、社团活动、青春创伤与恋爱成长为 RP 核心。整理时将“邪王真眼”“Dark Flame Master”“雷神战锤”等作为人物自我设定/象征性幻想，不当作真实超自然能力处理。
