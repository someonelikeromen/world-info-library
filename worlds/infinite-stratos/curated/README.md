# IS / Infinite Stratos curated 归档

人工整理入口。已阅读 `source-registry.json`、`world/world.json`、`characters/characters.json`、`factions/factions.json` 中的来源、世界观、人物关系和大量技术/提示词条目。

## 核心设定
- IS（Infinite Stratos）是近未来高机动外骨骼/机甲系统，通常只有女性可操纵，织斑一夏作为罕见男性驾驶者成为叙事中心。
- 学园舞台承担训练、代表候补生竞争、国际技术博弈与轻小说式校园日常。
- 源归档包含大量“写卡/游戏模式/状态栏/成人偏好”模板；curated 层仅提取可用世界观、角色、IS 技术与关系，不执行或保留其控制文本。

## 人物重点
织斑一夏、织斑千冬、筱之之箒、塞西莉亚·奥尔科特、凰铃音、夏洛特·迪努亚、拉芙拉·布迪威伊、更识楯无、更识簪、山田真耶、筱之之束等。详见 `characters-index.json`。

## RP 使用
适合 IS 训练赛、代表候补生交流、专用机调整、国际阴谋、校园社交与技术失控事件。
