# JOJO curated archive

手工整理对象：`jojo`。

本目录只整理既有归档中的世界观、替身体系、主线人物与组织关系；不改动 `raw/`、`imports/` 或自动归档文件。

## 标准产物

- `world.json`
- `source-registry.json`
- `characters-index.json`
- `knowledge-graph.json`
- `relationship-graph.json`
- `curation-notes.md`

## 整理重点

本世界以跨代家族传奇、替身能力、波纹/石鬼面/箭等规则为核心。curated 层将完整保留主线人物与敌对势力脉络，避免把原文中的说明性文本误当作人物。
