- 本能市/本能字学园是鬼龙院家在东京湾人工岛上的统治中心，学生星级决定生活待遇。
- 战斗生命纤维来自外星生命体，是极制服、神衣与 REVOCS 全球服饰控制的根源。
- 鬼龙院皋月以本能字学园与四天王集结力量，最终目标与鬼龙院罗晓/原始战斗生命纤维的计划相冲突。
- 流子、鲜血、断斩之剪构成寻找父亲真相与反击生命纤维阴谋的主线。
