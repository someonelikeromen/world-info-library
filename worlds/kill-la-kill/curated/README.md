# 斩服少女 / Kill la Kill curated 归档

人工整理入口。已阅读 `source-registry.json`、`world/world.json`、`characters/characters.json` 中的世界观、战斗生命纤维、人物和本能字学园相关条目。

## 核心设定
- 本能市/本能字学园是鬼龙院家在东京湾人工岛上的统治中心，学生星级决定生活待遇。
- 战斗生命纤维来自外星生命体，是极制服、神衣与 REVOCS 全球服饰控制的根源。
- 鬼龙院皋月以本能字学园与四天王集结力量，最终目标与鬼龙院罗晓/原始战斗生命纤维的计划相冲突。
- 流子、鲜血、断斩之剪构成寻找父亲真相与反击生命纤维阴谋的主线。

## 人物重点
缠流子、鬼龙院皋月、满舰饰真子、鬼龙院罗晓、针目缝、鲜血、蟇郡苛、猿投山涡、犬牟田宝火、蛇崩乃音、美木杉爱九郎、黄长濑紬等。详见 `characters-index.json`。

## RP 使用
适合学园等级冲突、极制服社团战、生命纤维阴谋、姐妹对峙、裸体海滩地下行动与决战 COVERS。
