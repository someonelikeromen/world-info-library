# 魔女之旅 curated 归档

本目录为人工整理的 RP 世界观入口，来源于 `source-registry.json` 指向的用户世界书，以及已阅读的 `characters/characters.json`、`world/world.json`、`factions/factions.json`、`locations/locations.json`、`reports/unclassified-entries.json` 等现有归档。

## 核心可用设定
- 旅行叙事：各国风俗、魔法师制度与旅途中短篇事件并重。
- 魔法职业阶梯：魔导士 → 魔女见习生 → 魔女；魔女以胸针与“魔女名”作为身份标识。
- 魔法统合协会：负责见习生考试、魔女派遣、犯罪制止、违禁品查封等事务。
- 重要舞台：魔法师之国、和平国罗贝塔、王立瑟雷斯特利亚、自由之城库诺兹、时钟乡罗斯特洛夫、深邃森林比拉、东方国度。

## 主要人物
伊蕾娜、沙耶、美奈、芙兰、席拉、维多利加、艾姆妮西亚、艾维莉亚、艾丝黛尔等。详见 `characters-index.json`。

## RP 使用提示
适合旅行见闻、魔女考试、协会任务、师徒关系、跨国委托、时间魔法与记忆代价等剧情。源文件存在部分角色向个人情感倾向描述，curated 层仅保留关系事实，不展开成人化内容。
