## 核心定位
现代魔法骑士学园战斗世界。具备魔力者被称为伐刀者，以灵魂具现为固有灵装，通过学园训练、校内选拔与七星剑武祭证明自身。主题集中于“最弱战胜最强”、技艺与意志对抗天赋、骑士荣誉与家族/国家/联盟政治压力。

## 主要舞台
- 破军学园：黑铁一辉、史黛菈、珠雫等人的主舞台。
- 七星剑武祭/湾岸巨蛋：全国七所骑士学园代表交锋的核心赛事。
- 晓学园：由月影总理设立、牵动日本政治与联盟秩序的国立学园势力。
- 法米利昂皇国、奎多兰王国：后期国际篇关联地点。
