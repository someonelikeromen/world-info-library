# 对魔忍 curated 归档

人工整理入口。已阅读 `source-registry.json`、`world/world.json`、`characters/characters.json` 中的世界设定、五车学园、井河阿莎姬、水城雪风、秋山凛子、高坂静流、水城不知火等条目。

## 核心设定
- 近未来日本，负面情绪与堕落区域引发连接魔界的空间裂隙，魔族入侵人类世界。
- 人类政府秘密组织“对魔忍”执行暗杀、情报、反恐、对魔族战斗等任务。
- 五车学园是对魔忍训练、选拔和任务部署的重要基地。
- 忍法体系包括水遁、雷遁、空遁、木遁、对魔杀法等，通常与个人血统、家族和战斗风格绑定。

## 人物重点
井河阿莎姬、水城雪风、秋山凛子、高坂静流、水城不知火，以及源中作为地点/组织核心的五车学园。详见 `characters-index.json`。

## RP 使用
适合五车学园训练、魔界裂隙调查、潜入任务、家族忍法传承、魔族追踪、失踪对魔忍救援等剧情。源内成人化控制文本、状态栏和玩家怪物设定未纳入 curated。
