## 世界定位

本世界以《魔法禁书目录》旧约早期至 9月30日前后事件为核心，兼容超炮相关学园都市角色。核心冲突为“科学侧”与“魔法侧”在同一时空下以不同法则运作：学园都市通过超能力开发与个人现实制造能力者；魔法侧由宗教、魔导书、圣人、术式与秘密组织构成。上条当麻的幻想杀手连接并冲突于两侧。
