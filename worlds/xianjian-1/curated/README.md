# 仙剑奇侠传Ⅰ curated 归档

本目录为 `xianjian-1` 的人工整理产物，来源于既有归档文件，重点提炼为 RP 可用的世界观索引。

## 标准产物

- `world.json`：世界总览、地点、势力、战力与剧情钩子。
- `source-registry.json`：本 curated 批次引用的源登记。
- `characters-index.json`：详细人物清单。
- `knowledge-graph.json`：概念、地点、势力、物品与剧情节点。
- `relationship-graph.json`：人物与组织关系。
- `curation-notes.md`：人工整理说明与风险。

## 人工整理重点

- 余杭镇、仙灵岛、苏州、蜀山、锁妖塔、南诏国构成主线地理轴。
- 女娲族、蜀山派、拜月教、黑苗/白苗、林家堡构成主要势力冲突。
- 战力采用原归档中的凡人境、后天境、先天境、筑基境、金丹境、元婴/化神境、地仙/天仙境、神魔境层级。
- 主要剧情围绕李逍遥、赵灵儿、林月如、阿奴对抗拜月教主与水魔兽，同时保留 RP 分歧可能。
